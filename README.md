# Finance Analytics Dashboard

An interactive Power BI dashboard designed to analyze personal income, expenses, and savings over time. This project helps users understand spending patterns, track savings growth, and make informed financial decisions using data-driven insights.


# Project Overview

The **Finance Analytics Dashboard** provides a comprehensive view of financial performance by visualizing income, expenses, and savings across multiple years and categories. It enables effective budgeting, expense control, and long-term financial planning.



# Objectives

- Track total **Income, Expenses, and Savings**
- Analyze **Expense % and Saving %**
- Monitor **Month-on-Month (MoM) financial trends**
- Identify major **spending and saving components**
- Support better **personal financial planning**

# Tech Stack

- **Power BI Desktop** – Dashboard creation and data visualization  
- **Power Query** – Data cleaning, transformation, and preprocessing  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **Data Modeling** – Relationships between income, expense, savings, and date tables  
- **File Formats**  
  - `.pbix` – Dashboard development file  
  - `.png` – Dashboard preview images  

# Data Source

**Source:**  
- Manually curated personal finance dataset  

**Data Includes:** 
- Year and Month  
- Income sources (Salary, Other Sources)  
- Expense categories (Rent, EMI, Groceries & Food, Health, Shopping, Leisure)  
- Savings components (Mutual Funds, Fixed Deposit, Emergency Fund, Liquid Cash)


# Dashboard Features

# Key KPIs
- Expense %
- Saving %
- Total Expense
- Total Income
- Total Savings

# Filters
- Year-wise slicer (2018–2021)

# Visualizations
- Income, Expense & Saving **MoM Trend Line**
- Expense distribution by component
- Savings distribution by component
- Detailed monthly financial table


# Business Problem

Without clear financial insights, individuals often overspend and fail to save efficiently. Tracking expenses manually makes it difficult to identify spending leaks and savings opportunities.



# Dashboard Goals

- Provide a **clear financial summary**
- Highlight **high-expense categories**
- Track **savings growth and allocation**
- Improve **financial awareness and discipline**



# Key Insights

- Rent and EMIs are the highest expense contributors  
- Mutual Funds contribute the largest share of savings  
- Saving percentage shows consistent improvement over time  
- Month-on-month trends help identify income and expense fluctuations  



# Key Questions Answered

- What percentage of income is spent versus saved?
- Which categories contribute most to expenses?
- How do income and expenses change month-on-month?
- Are savings growing consistently?
- Which saving instruments are most effective?
